# Настройка Google Analytics и Яндекс.Метрики

## 📊 Обзор

В проект интегрированы системы аналитики Google Analytics 4 и Яндекс.Метрика с поддержкой многоязычности и отслеживанием пользовательских событий.

## 🔧 Настройка

### 1. Google Analytics 4

1. Создайте аккаунт Google Analytics 4 на [analytics.google.com](https://analytics.google.com)
2. Получите ваш измерительный ID (начинается с `G-`)
3. Замените `G-XXXXXXXXXX` в следующих файлах:
   - `index.html` (строки 42 и 52)
   - `src/components/Analytics.tsx` (строка 6)

### 2. Яндекс.Метрика

1. Создайте счетчик на [metrica.yandex.ru](https://metrica.yandex.ru)
2. Получите ваш номер счетчика (числовой ID)
3. Замените `12345678` в следующих файлах:
   - `index.html` (строки 60 и 105)
   - `src/components/Analytics.tsx` (строка 7)

## 📈 Отслеживаемые события

### Автоматические события
- **Просмотры страниц** с указанием языка
- **Смена языка** (с какого на какой)
- **Глубина прокрутки страницы**

### Пользовательские события
- **Клики по CTA кнопкам** (`button_click`)
- **Генерация лидов** (`lead_generated`)
  - Тип: phone, email, form
- **Просмотр секций** (`section_view`)

## 🌍 Многоязычная поддержка

Система автоматически отслеживает:
- Язык интерфейса пользователя
- Переходы между языковыми версиями
- Отдельную статистику для каждого языка

## 🎯 Настройка целей

### Google Analytics
Рекомендуемые конверсии для настройки:
1. **Звонки** - событие `lead_generated` с `lead_type: phone`
2. **Заявки** - событие `lead_generated` с `lead_type: form`
3. **Глубокое изучение** - событие `scroll_depth` > 75%

### Яндекс.Метрика
Рекомендуемые цели:
1. **Звонок** - цель `lead_generated`
2. **Заявка** - цель `button_click`
3. **Изучение каталога** - цель `section_view`

## 📊 Кастомные события

Для добавления собственных событий используйте:

```typescript
import { trackEvent } from '@/components/Analytics';

// Пример отслеживания
trackEvent('custom_event', {
  category: 'user_action',
  language: 'ru',
  custom_parameter: 'value'
});
```

## 🔒 Конфиденциальность

- Включена анонимизация IP-адресов
- Настроено время жизни cookies (2 года)
- Соблюдение GDPR через настройки согласия

## 📱 Мобильная аналитика

Система отслеживает:
- Тип устройства (desktop/mobile/tablet)
- Размер экрана
- Touch-события на мобильных устройствах

## 🚀 Производительность

- Асинхронная загрузка скриптов
- DNS prefetch для быстрой загрузки
- Отложенная инициализация для Яндекс.Метрики
